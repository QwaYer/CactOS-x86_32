🚀 Core Features

🧠

Memory Management

PMM, VMM, heap, slab allocator, mmap, COW, swap, page faults, OOM handling, shared memory.

⏱️

MLFQ Scheduler

4-level multilevel feedback queue with priority boost, anti-starvation, and sleep queue.

💾

Multiple Filesystems

ext4, VFS, devfs, procfs, mntfs, etcfs, tmpfs, binfs, sbinfs, libfs, and varfs.

🌐

Complete Network Stack

Ethernet, ARP, IPv4, ICMP, TCP/UDP sockets, smoltcp, virtio-net, DNS, DHCP.

🔌

Driver Support

PCI, AHCI, NVMe, xHCI, USB, PS/2, virtio-net, framebuffer console with MTRR.

🔒

Synchronization

Spinlock, IRQ spinlock, mutex, semaphore for multi-process coordination.

🧬

Process Management

fork, exec, exit, waitpid. 13 signals, sigaction, full process lifecycle support.

📦

ELF & Dynamic Linking

Complete ELF loader, dynamic linking, i386 relocations including R_386_COPY.

🎯

Multiboot2 Boot

Full Multiboot2 support with framebuffer, memory map, modules, and PCI config.

🏗️ Building CactOS

Quick Start (one command):

./build-cact-qemu.sh # full ISO + disk
RUN_QEMU=1 ./build-cact-qemu.sh # build + run in QEMU

Step by step:

make -j$(nproc) # full ISO (libc → shell → utils → kernel)
make -j$(nproc) disk # ISO + empty ext4 nvme.img
make -j$(nproc) kernel # kernel only
make -j$(nproc) libc # libc only
make -j$(nproc) cactsole # shell only

Fine-grained targets:

make -C CactLib-x86_32 # libc only
make -C Cactsole-x86_32 # shell only
make -C CactUserBins-x86_32 install # userland utils
make -C CactKernel-x86_32 iso-full # kernel + userland
make -C LocalRepoCactOS # pack cctkfs.img
make -C CactBridge iso # final ISO

Requirements:

• gcc -m32 (multilib) for 32-bit compilation
• nasm for NASM assembly
• cargo +nightly for Rust subsystems
• grub-mkrescue + xorriso for ISO creation
• qemu-img, mkfs.ext4 for disk images
• make for build automation

Build with options:

SKIP_DRIVERS=1 make # skip out-of-tree drivers
DRIVERS="AHCI NVMe" make # build specific drivers
JOBS=8 make # parallel build with 8 jobs

🗺️ System Architecture

Workspace structure integrating all components:

CactOS-x86_32/
├── CactKernel-x86_32/ # Core kernel
├── CactLib-x86_32/ # C library
├── Cactsole-x86_32/ # Shell
├── Cgoct-x86_32/ # Init process
├── CactUserBins-x86_32/ # User utilities
├── LocalRepoCactOS/ # Module repository
├── CactBridge/ # ISO builder
├── build-cact-qemu.sh # Full build script
└── Makefile # Workspace build
