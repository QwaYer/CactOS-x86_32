🔍 Technical Specifications

🧠

Memory Manager

PMM, VMM, heap, slab, mmap, COW, swap, page faults, OOM handling, shared memory (Rust: rust_mm)

⏱️

Scheduler

4-level MLFQ (Rust: sched) with priority boost, anti-starvation, sleep queue, and timer wheel.

🔒

Synchronization

Spinlock, IRQ spinlock, mutex, semaphore (Rust implementation) for multi-process coordination.

🧬

Processes & Signals

TaskStruct, fork/exec/exit/waitpid, 13 signals, sigaction, sigreturn, sigprocmask, sigsuspend.

📦

ELF & Dynamic Linking

ELF loader, dynamic linker, i386 relocations including R_386_COPY for complete binary support.

📁

VFS & Filesystems

VFS core, ext4 read/write, devfs, procfs, mntfs, etcfs, tmpfs, binfs, sbinfs, libfs, varfs.

💾

Block I/O & Drivers

PCI enumeration, AHCI, NVMe, block cache, xHCI, USB, PS/2, virtio-net, framebuffer console.

🌐

Network Stack

Ethernet, ARP, IPv4, ICMP, TCP/UDP sockets, smoltcp (Rust), virtio-net, DNS, DHCP, knetd.

🎯

Boot & System

Multiboot2 parser, GDT/TSS, IDT (32 CPU exceptions + PIC IRQs), framebuffer, version tracking.

🔗 Integrated Ecosystem

🏗️ Building CactOS

Quick Start - Full ISO + Disk:

./build-cact-qemu.sh # full ISO + disk, 1 command
./build-cact-qemu.sh && CACT_ISO=CactBridge/build/cact.iso CactKernel-x86_32/run_qemu.sh
RUN_QEMU=1 ./build-cact-qemu.sh # build + run in QEMU

Or Step by Step from Workspace:

make -j$(nproc) # full ISO (default)
make -j$(nproc) disk # ISO + empty nvme.img
make -j$(nproc) kernel # kernel only
make -j$(nproc) libc # libc only
make -j$(nproc) cactsole # shell only
make -j$(nproc) drivers # all out-of-tree drivers

Build Individual Components:

make -C CactLib-x86_32 # libc (no deps)
make -C Cactsole-x86_32 # shell (auto-finds CactLib)
make -C Cgoct-x86_32 # init (auto-finds CactLib)
make -C CactUserBins-x86_32 install # userland utils
make -C CactKernel-x86_32 # kernel only ISO
make -C CactKernel-x86_32 iso-full # kernel + userland
make -C LocalRepoCactOS # pack cctkfs.img
make -C CactBridge iso # final ISO from kernel + cctkfs

Requirements:

• gcc -m32 (multilib on amd64 — e.g. gcc-multilib)
• nasm — Multiboot2 entry + interrupt stubs
• ld -m elf_i386 — GNU binutils
• cargo +nightly — Rust subsystems (MM, scheduler, net)
• grub-mkrescue + xorriso — ISO creation
• qemu-img, mkfs.ext4, e2fsck — Disk image management
• make — Build automation

Build Options:

SKIP_DRIVERS=1 make # skip out-of-tree drivers
DRIVERS="AHCI NVMe Virtio-net" make # build specific drivers only
JOBS=N make # parallel build with N jobs
